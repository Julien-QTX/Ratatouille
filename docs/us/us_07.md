# US 7 - Mode hors-ligne
- [x] Scan produit connu en mode avion affiche la fiche locale
- [x] Indicateur 'Données hors-ligne'
- [x] Historique et favoris dispo offline
- [x] Scan inconnu offline affiche erreur spécifique

## Implémentation Technique
- **Offline-First** : `ProductRepository` cherche d'abord le produit sur le réseau. Si échec (`UnknownHostException`), il cherche dans la base `Room`.
- **Indicateur** : Le `Product` domaine possède un flag `isOffline` setté à `true` lors de la conversion de l'Entity locale. La `ProductCard` affiche l'icône `CloudOff` si ce flag est vrai.
