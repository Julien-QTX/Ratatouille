# US 10 - Partage d'un produit via deep link
- [x] Bouton 'Partager' sur la fiche générant 'myapp://product/{id}'
- [x] Ouverture du lien ouvre la fiche produit dans l'app
- [x] Téléchargement depuis l'API si non local
- [x] Message d'erreur explicite si lien invalide

## Implémentation Technique
- **Deep Links** : Déclaration de `<data android:scheme="myapp" android:host="product" />` dans l'`AndroidManifest.xml` sur l'Activity principale.
- **Navigation Compose** : Ajout du paramètre `deepLinks` sur la route `Screen.Scan` dans `NavHost`.
- **Partage** : Utilisation d'un `Intent.ACTION_SEND` (texte brut) appelé via un `IconButton` "Share" sur la `ProductCard`.
