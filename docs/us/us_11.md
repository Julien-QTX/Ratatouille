# US 11 - Widget écran d'accueil
- [x] Widget ajoutable sur l'écran d'accueil
- [x] Affiche nom et NutriScore du dernier scan
- [x] 'Aucun scan récent' si vide
- [x] Appui ouvre l'app (ScanScreen)
- [x] Mise à jour auto après nouveau scan

## Implémentation Technique
- **Outil** : Librairie **Jetpack Glance** (`glance-appwidget`).
- **Composant** : Création de `RatatouilleWidget` (UI en code Compose-like) et `RatatouilleWidgetReceiver`.
- **Déclaration** : Ajout du `<receiver>` et du XML de métadonnées dans le Manifest.
- **Mise à jour** : Le `ProductRepository` appelle `RatatouilleWidget().updateAll(context)` (dans une coroutine) à chaque fois qu'un produit est inséré ou supprimé de l'historique pour garder le widget synchronisé.
