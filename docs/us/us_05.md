# US 5 - Supprimer de l'historique
- [x] Swipe pour supprimer
- [x] Confirmation demandée
- [x] Disparaît immédiatement
- [x] Ne supprime pas des favoris

## Implémentation Technique
- **UI** : Utilisation de `SwipeToDismissBox` (API Material 3 expérimentale) sur les éléments de liste.
- **Logique de suppression** : Boîte de dialogue de confirmation (`AlertDialog`).
- **Préservation Favoris** : Si le produit est favori, `deleteProduct` met son `scanDate` à 0 au lieu d'un `DELETE` SQL. Le DAO d'historique exclut les `scanDate <= 0`.
