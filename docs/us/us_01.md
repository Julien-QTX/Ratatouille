# US 1 - Fiche produit (scan)
- [x] Scan code-barres
- [x] Fiche avec nom, marque, NutriScore coloré, image
- [x] Message d'erreur clair si introuvable
- [x] Message d'erreur clair si hors-ligne
- [x] Code couleur NutriScore correct

## Implémentation Technique
- **Réseau** : `Retrofit2` et `Moshi` pour requêter l'API OpenFoodFacts (`/api/v0/product/{barcode}.json`).
- **UI** : `Jetpack Compose` avec un composant réutilisable `ProductCard`.
- **Images** : Librairie `Coil` (`rememberAsyncImagePainter`) pour le chargement asynchrone de l'image.
- **Gestion des Erreurs** : Interception des exceptions `HttpException` (ex: 404 renvoyant "Produit non trouvé") et `UnknownHostException` pour le mode hors-ligne dans le `ProductRepository`.
