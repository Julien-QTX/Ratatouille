# US 4 - Ajouter favori
- [x] Bouton étoile sur fiche produit
- [x] Ajout/Retrait direct
- [x] Ecran Favoris trié par date
- [x] Persistance
- [x] Peut être historique ET favoris

## Implémentation Technique
- **Modèle de données** : Ajout d'un booléen `isFavorite` et d'un Long `favoriteDate` dans `ProductEntity`.
- **Dao** : Requête `SELECT * FROM products WHERE isFavorite = 1 ORDER BY favoriteDate DESC`.
- **UI** : Toggle avec `Icons.Default.Star` / `StarBorder`. Un `FavoritesViewModel` expose le flux à `FavoritesScreen`.
