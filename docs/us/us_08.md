# US 8 - Sync arrière-plan
- [x] Tâche WorkManager 1x/jour
- [x] Contrainte réseau
- [x] Retry auto
- [x] Fonctionne app fermée
- [x] UniqueWork

## Implémentation Technique
- **Outil** : Utilisation d'`androidx.work:work-runtime-ktx` (WorkManager).
- **Worker** : Création d'un `SyncFavoritesWorker` (héritant de `CoroutineWorker`) qui boucle sur `getFavoritesList()` et appelle l'API pour chaque produit.
- **Planification** : Enregistrement dans `RatatouilleApplication` d'une `PeriodicWorkRequest` (1x/jour) avec `NetworkType.CONNECTED` et une politique de retry `BackoffPolicy.LINEAR`.
